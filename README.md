# Project Management System

A comprehensive project management solution featuring:
- **Client**: React-based frontend with workspace management, project tracking, and task boards.
- **Backend**: Node.js/Express server handling authentication, database operations, and API endpoints.

## Features
- User Authentication (Email & Google OAuth)
- Workspace & Project Organization
- Task Management with Kanban/List views
- Team Collaboration tools

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation

1. Clone the repository
2. Install dependencies for both client and backend:
   ```bash
   cd client && npm install
   cd ../backend && npm install
   ```
3. Configure environment variables in `.env` files.
4. Run the development servers:
   ```bash
   # Terminal 1
   cd client && npm run dev

   # Terminal 2
   cd backend && npm run dev
   ```
